# Stone Strategy Changelog

## stone_0.4.7 — 2026-07-07

0.4.5 + 最低回调深度过滤（MIN_PULLBACK_PCT = 0.015）。

### 核心变化（vs 0.4.5）
- **MIN_PULLBACK_PCT = 0.015**: 跳过回调深度 < 1.5% 的入场
  - 浅回调 → 三档目标价扎堆 → 扣除滑点后利润为负
  - 3%阈值过于激进（过滤掉ABSI +$119等好交易），1.5%为折中值
  - 仅影响首笔交易入场判断，再入场逻辑不变

### 90天回测结果（$1,000启动资金，含滑点）
| 指标 | 0.4.5 | 0.4.7 |
|------|-------|-------|
| 交易数 | 189 | 145 |
| 胜率 | 69.3% | 71.0% |
| 总P&L | $58,297 | $31,910 |
| 盈利因子 | 5.94 | 5.81 |
| 最大亏损 | -$1,871 | -$1,317 |

### 被过滤交易分析（50笔，P&L = +$9,512）
| 回调深度 | 交易数 | 胜率 | P&L |
|---------|--------|------|------|
| 负值（entry>open） | 19 | 31.6% | +$2,848 |
| 0-0.5% | 13 | 76.9% | +$1,711 |
| 0.5-1% | 15 | 86.7% | +$2,956 |
| 1-1.5% | 3 | 33.3% | +$1,998 |

**结论: 1.5%过滤仍过于激进。浅回调交易整体盈利，过滤后P&L下降45%。0.4.5仍为最佳版本。0.4.7作为低风险备选（最大亏损更小）。**

## stone_0.4.6 — 2026-07-05

0.4.5 + 低价股过滤 + 缺口上限。

### 核心变化（vs 0.4.5）
- **PRICE_MIN 从 $0.25 提升至 $1.00**: 排除亚美元股票（防pump-and-dump）
- **GAP_MAX = 0.30**: 排除缺口超过30%的极端gap（如UZX +40.8%）

- 总P&L: $65,127
- 胜率: 63.3%

- **结论: GAP_MAX=30%过于激进，过滤掉大量高质量高缺口交易，P&L从$548K降至$65K。不如0.4.5**

## stone_0.4.5 — 2026-07-05 (当前最佳版本)

- 移动止损: 3%/4%/5%
- 再入场: 5%止损, 150%目标卖1/3, 之后5%移动止盈, 13:00截止

## stone_0.4 — 2026-06-25 (当前最佳版本)

三档止盈 + 量价确认再入场。在0.3基础上增加112.5%中间止盈档和回撤再入场机制。

### 核心变化（vs 0.3）
- **三档止盈**: 75%卖1/4 + 112.5%卖1/3 + 150%卖1/3，替代0.3的两档
- **量价确认再入场**: 首笔交易出场后，冲顶回撤时需满足量价齐升才可再入场
  - 确认bar必须收阳（close > open）
  - 确认bar成交量 > 近5根bar平均量的1.2倍
  - 再入场止损5%，目标150%回撤卖1/3，之后5%移动止盈
- **大幅回调保护**: 从日内高点回撤超15%则停止当日交易
- **每日最多3笔交易**（首笔+再入场合计）

### 参数
- 首笔交易：同0.3（ATR 2.0止损, 3%/4%/5%三档移动止盈）
- GAP_THRESHOLD: 0.20
- EQUITY_POSITION_RATIO: 0.80
- MAX_POSITION_SIZE: 100,000
- MIN_POSITION_SIZE: 250
- INITIAL_CAPITAL: 1,000
- PARTIAL_SELL_RATIO_75: 1/4 / 1125: 1/3 / 150: 1/3
- TRAILING_STOP_PCT_75: 0.03 / 1125: 0.04 / 150: 0.05
- REENTRY_STOP_PCT: 0.05 (5%止损)
- REENTRY_PROFIT_RETRACEMENT: 1.50 (150%目标)
- REENTRY_SELL_RATIO: 1/3
- REENTRY_TRAILING_PCT: 0.05
- PULLBACK_STOP_THRESHOLD: 0.15 (15%大幅回调停止)
- MAX_DAILY_TRADES: 3

### 180天回测结果（$1,000启动资金）
- 最终资金: $616,069
- **总收益率: 61,507%**
- 胜率: 80.5%
- 最大回撤: -14.74%
- 夏普比率: 14.46
- 总交易: 215笔 (首笔166 + 再入场49)
- 首笔胜率: 87.3% | 再入场胜率: 57.1%
- 再入场止损仅8次（无条件版30次）

## stone_0.3 — 2026-06-24

复利仓位 + 动态部分止盈。取消跳空分级，统一参数。

### 核心变化（vs 0.1）
- **复利仓位**: 仓位 = 当前权益 × 80%（最低$250），随盈利自动放大
- **动态部分止盈**: 75%卖出1/4 + 150%卖出1/3剩余，替代原来150%卖出1/3
- **取消跳空分级**: 所有>20%跳空统一参数（ATR 2.0, 3%/5%移动止盈）
- **单股仓位上限**: $100K，超出则增加持股数量
- **保留**: 10:00前入场时间窗口、入场确认、流动性过滤

### 参数
- GAP_THRESHOLD: 0.20
- EQUITY_POSITION_RATIO: 0.80
- MAX_POSITION_SIZE: 100,000
- MIN_POSITION_SIZE: 250
- INITIAL_CAPITAL: 1,000
- PARTIAL_SELL_RATIO_75: 1/4 / 150: 1/3
- TRAILING_STOP_PCT_75: 0.03 / 150: 0.05
- STOP_LOSS_ATR_MULT: 2.0
